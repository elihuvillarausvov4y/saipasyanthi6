## Introduction 

A XMake integration in Visual Studio Code. 

Please see [xmake-github](https://github.com/tboox/xmake) and [website](http://xmake.io) if you want to known more about xmake.

## Features

* Colorization
* Completion Lists
* StatusBar
* Commands
* Configuration
* Build
* Debug
